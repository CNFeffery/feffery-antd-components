Subproject of `feffery-components` projects, third-party components library of `Plotly Dash`, based on the [Antd](https://github.com/ant-design/ant-design), brings many common and powerful components into `Dash`ðŸ¥³, latest stable version: `0.3.8`

## 1 Install

```bash
pip install feffery-antd-components -U
```

## 2 Install the latest preview release version

> [!NOTE]  
> The latest preview release version (2024-08-09): `0.3.0rc31`

```bash
pip install feffery-antd-components --pre -U
```

## 3 Use CDN

```Python
# just set serve_locally=False in dash.Dash()
app = dash.Dash(serve_locally=False)
```
